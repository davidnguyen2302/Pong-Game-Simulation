This is a simulation of a pong game (player 1: top, player 2: bottom)
The game ends when one of the player reaches the score of 5 (although it can be changed via slider)
Speed and spawn direction of the ball is randomized at the start and every time a player scores.
Press "SAVE" to save state of the game, "LOAD" to load the saved state, "RESTART" to start over.
